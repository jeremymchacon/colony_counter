Colony_Counter

By: Jeremy M. Chacon
contact:  chaco001 at you em en .edu

Colony_Counter is a repo for counting bacterial colonies using python and skimage. Right now the one
script present (count_colonies.py) can count light-colored colonies on a dark background, in a petri dish 
centered in a square image. 

Usage: python count_colonies.py [imagename.tif]
